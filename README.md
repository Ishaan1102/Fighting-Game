# Fighting Game

This is a simple fighting game built with HTML, CSS, and JavaScript.

## How to Play
  Plater 1:
- Use the arrow keys to move your character.
- Use down key to attack
  Player 2:
- Press the spacebar to attack.
- Defeat all enemies to win the game.

## Technologies Used

- HTML
- CSS
- JavaScript
